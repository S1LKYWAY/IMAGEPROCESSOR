# Image Processing Qt Application

## Author: Qasim Abbas

### Overview

This Qt application is designed for image processing tasks. The application allows users to open an image file (JPEG, JPG, PNG), and then performs a series of image transformations in sequential steps.

### Features

1. **Step 1: Colored and Grayscale**
   - Displays the original colored image on the left.
   - Displays the grayscale version of the colored image on the right.

2. **Step 2: Colored and Darkened**
   - Displays the original colored image on the left.
   - Displays a darkened version of the colored image on the right.

3. **Step 3: Grayscale and Dithered**
   - Displays the grayscale version of the original colored image on the left.
   - Displays a dithered image of the grayscale version on the right.


### Usage

1. Run the application.
2. Click on "File" in the menu bar and select "Open File" to choose an image.
3. The application will display the image in the specified steps.
4. Click "Next" to progress through the steps.

### Dependencies

- Qt 5 or later.

### Build and Run

1. Ensure you have Qt installed on your system.
2. Open the project in Qt Creator or your preferred Qt development environment.
3. Build and run the project.
